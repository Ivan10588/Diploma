# API сохранённых поисков

## POST /equipment/api/save-search/
Сохраняет новый поисковый запрос.

**Параметры:**
* `name` (обязательный, строка, до 100 символов) — название поиска;
* `filters` (обязательный, JSON‑объект) — параметры поиска в формате JSON;
* `notify` (опционально, boolean, по умолчанию `true`) — получать уведомления о новых объявлениях;
* `frequency` (опционально, строка, значения: `daily`/`weekly`, по умолчанию `daily`) — частота уведомлений.

**Примеры тела запроса:**

1. Базовый вариант:
```json
{
  "name": "Экскаваторы до 5 млн",
  "filters": {
    "equipment_type": "excavator",
    "price_max": 5000000
  }
}
